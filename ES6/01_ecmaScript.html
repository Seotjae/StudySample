ECMA SCRIPT 란?

ECMA 인터내셔널에 의해 제정된 ECMA-262 기술 규격에 의해 정의된 범용 스크립트 언어

JAVA SCRIPT 는 브라우저마다 해석이 달라 동작이 달랐다.

그래서 이에 대한 표준을 세운것이 바로 ECMA-262, 즉 ECMA SCRIPT 이다.

현재 우리가 사용하는 자바스크립트는 ECMA SCRIPT 5 버전 (2009.12)

ES6(2015) 이후 부터는 년도를 붙이게 되었는데 이를테면 ES7 은 ES2016으로 불리는 것이다.

최신 버전은 ES2018 이다.

ECMA SCRIPT 최신 버전은 어디에 사용되나요?

JAVA SCRIPT 로 구현된 최신 프레임워크들에서는 사용한다.

하지만 최신 브라우저가 아닌 경우 제대로 동작하지 않아 PC용 웹에서는 아직은 사용하지 않는다.

모바일 전용 웹에서는 사용할 수 있다. (폰은 10년 동안 들고있는 경우가 거의 없으므로)

기존의 JAVA SCRIPT 에 대해 이해한 후 ES6 공부를 하는 것이 좋다.
